# Operatorii `<`, `>`, `<=`, `=>`

Evaluarea acestor operatori conduce la un răspuns `true` sau `false`.

## Mai mare `>`, mai mic `<`

Testează valorile după ordinul de mărime. `<` înseamnă testează valoarea din stânga să fie mai mică decât cea din dreapta. `>` este taman invers.

```javascript
5 > 3; // true
18 < 12; // false
```
