# Semnele de punctuați folosite de ECMAScript

Lista semnelor de punctuație folosite:

`{`, `(`, `)`, `[`, `]`, `.`, `...`, `;`, `,`, `<`, `>`, `<=`, `>=`, `==`, `!=`, `===`, `!==`, `+`, `‐`, `*`, `%`, `++`, `‐‐`, `<<`, `>>`, `>>>`, `&`, `|`, `^`, `!`, `~`, `&&`, `||`, `?`, `:`, `=`, `+=`,`‐=`, `*=`, `%=`, `<<=`, `>>=`, `>>>=`, `&=`, `|=`, `^=`, `=>`, `**`, `**=`, `/`, `/=`, `}`.
